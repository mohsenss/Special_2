# Steel Surge (Godot 4.6)

لعبة 3D منظور شخص أول مبنية خصيصًا لتكون متوافقة مع **Godot 4.6**.

## التشغيل
1. افتح المشروع من Godot 4.6 عبر ملف `project.godot`.
2. شغّل المشهد الرئيسي `scenes/Main.tscn` (مضبوط تلقائيًا كمشهد بدء).

## التحكم
- **WASD**: حركة
- **Mouse**: توجيه الكاميرا
- **Space**: قفز + دبل جمب
- **Left Mouse**: ضربة سكين جماعية (AoE أمام اللاعب)
- **Shift**: Dash هجومي (يصدم ويضر الأعداء)
- **E**: اختفاء مؤقت (Stealth)
- **R**: إعادة اللعب بعد الخسارة
- **Esc**: إظهار الماوس

## المزايا الأساسية
- حلقة لعب واضحة: بقاء + قتال + كثافة متزايدة.
- نظام حركة سريع ومريح (قفز/دبل جمب/داش).
- قتال موجّه للحشود (Knife cleave + Dash impact).
- اختفاء تكتيكي يوقف استهداف الأعداء مؤقتًا.
- نظام Spawn كثيف مع سقف أعداء للحفاظ على الأداء.
- واجهة واضحة: صحة، قتلات، تبريد القدرات، Game Over.

## هيكلة المشروع
- `scenes/`: المشاهد الأساسية (Main, Player, Enemy, UI)
- `scripts/`: منطق الأنظمة (اللاعب، العدو، السباونر، واجهة المستخدم، مدير اللعبة)

## ملاحظات أداء
- AI مبسّط وفعّال مناسب للحشود الكبيرة.
- سقف أعداء حيّ (`max_alive`) يمكن تعديله من `EnemySpawner`.
- ضبط القيم قابل للتعديل بسهولة عبر `@export`.
